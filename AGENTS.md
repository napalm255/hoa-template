# HOA Template

This is a template for a HOA (Homeowners Association) website. For the sake of the site demo, let's use lorem ipsum text for the content.

The website will have a clean and modern design, with easy navigation and a responsive layout that works well on both desktop and mobile devices.

This is a github site using a CMS (Content Management System) to manage the content of the website. The CMS allows the HOA to easily update and maintain the website without needing technical knowledge.

## Features
- Home page with welcome message and news updates
- About Us page with information about the HOA and its mission
- Events page with upcoming events and activities
- Board of Directors page with information about the board members and their roles
- Newsletters
- Reminders/Tips page with important information for residents
- Parks page for information on community parks and amenities
- Committe Pages for each committee with information about their purpose and members
  - Welcome Committee
  - Architectural Committee
  - Recreation Committee
  - Pool Committee
- Facilities page with information about community facilities and how to reserve them
- Trash, Recycling, and Yard Waste page with information about waste management services and schedules
- Projects
- Reporting Problems
  - Suspicious Activity or Vandalism
  - Towing and Parking
  - Noisy Neighbors / Disturbances
  - Animal Control
  - Snow Removal
  - Postal Service and Concerns
  - Community Concerns
- Community Links page with links to local resources and services
  - Animal Control & Adoption
  - Churches
  - Community Services
  - Government
  - Libraries
  - News Media
  - Parks
  - Schools
  - Utilities
- Admin page for managing the website content.
- Easily update and replace the title, description, and content of each page through the CMS.

## Technologies
- HTML/CSS for the structure and styling of the website
- Reactive website for a smooth user experience
- CMS for easy content management
- GitHub Pages for hosting the website
- Admin page secured by github authentication for authorized access to manage the website content.
- Hugo and DecapCMS for the CMS and static site generation.

## Documentation

- Always updated `README.md` file with instructions on how to set up and manage the website.
- Detailed documentation on how to use the CMS and manage the website content.
- Troubleshooting guide for common issues that may arise when managing the website.
- Contact information for support and assistance with the website.
